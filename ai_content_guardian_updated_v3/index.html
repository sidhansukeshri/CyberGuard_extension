Features Overview

Key Features

  • Real-time Content Analysis: Scans webpage content as you browse using a BART AI model.
  • Multiple Detection Categories: Identifies harmful, offensive, and inappropriate content.
  • Visual Warning System: Adds non-intrusive warning badges to flagged content.
  • Automatic Content Rephrasing: Replaces harmful content with safer alternatives.
  • Customizable Settings: Adjust sensitivity levels and choose which protections to enable.
  • Performance Optimized: Uses local detection as a fallback when AI services are unavailable.
  • Usage Statistics: Tracks how many pieces of content have been analyzed and modified.

How It Works

  1. The extension scans text elements on webpages as they load.
  2. Text is sent to a BART AI model for analysis (or processed locally).
  3. Content categorized as harmful, offensive, or inappropriate is flagged.
  4. Based on user settings, the extension either adds warnings or rephrases content.
  5. Modified content is highlighted with subtle visual indicators.
  6. The extension caches results to avoid re-analyzing the same content.

Privacy & Performance

The extension prioritizes user privacy and browsing performance:

  • Analysis is performed on the user's device when possible.
  • When using AI services, only text snippets are transmitted, never personal data.
  • The extension uses throttling to avoid impacting page load times.
  • Users can disable the extension on specific sites or temporarily with one click.
